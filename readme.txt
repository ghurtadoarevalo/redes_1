Instrucciones para ejecutar laboratorio 1

Requisitos m√≠nimos:
	Python 3.*
	numpy
	matplotlib
	scipy

Para instalar las librerias se hace uso pip3, en donde:

	*Para instalar pip3
		[macOS] brew install pip3
		[ubuntu] apt install pip3
	*Para instalar numpy
		pip3 install numpy
	*Para instalar matplotlib
		pip3 install matplotlib
	*Para instalar scipy
		pip3 install scipy

* Omitir los pasos anteriores, en el caso de tenelos previamente instalados *

Se puede ejecutar de dos formas el codigo

1. Directamente desde consola:
	a.	Abrir terminal/consola
	b.	Posicionarce en el directorio del archivo "laboratorio_1.py"
	c.	ejecutar: python3 ./laboratorio_1.py
2.	Usando el IDLE Pycharm:
	a.	Abrir aplicacion Pycharm
	b. 	Selecionar Open, indicando donde se encuentra laboratorio_1.py
	c.	Una vez cargado el main, se debe precionar el boton verde de la derecha superior "run"
